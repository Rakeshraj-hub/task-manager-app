## How to Run Backend:-
  * cd Backend
  * npm install
  * npm start

## How to run Frontend
  * cd frontend
  * npm install
  * npm start

 ## ABOUT PROJECT
  A simple and clean Task Manager Application built using the MERN stack (MongoDB, Express.js, React.js, Node.js).
  Users can create, view, update, and delete tasks easily with validations and real-time UI updates.

  ## Tech Stack Used:-
   FRONTEND :- React.js, Tailwind Css, React Router, Zod(For Validation).
   BACKEND :- Node, Express, Mongoose, MongoDB Atlas
   TOOLS :- Postman, dotenv, nodemon

 ## Features:-
  Core Features:- 
         * Add new tasks
         * View all tasks
         * Edit/update task
         * Delete task
         * Real-time UI update
         * Status system: Pending, Running, Completed, Failed
         * Form validations using Zod
        * Error handling on UI
        * Automatic timestamps (createdAt, updatedAt)

  Bonus Features
       *Badge colors based on status
       * Human readable dates
       * Loading states
       * Modular components
       * Fully responsive UI
